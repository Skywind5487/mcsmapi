search(username: str = '', page: int = 1, page_size: int = 20, role: str = '') → mcsmapi.models.user.SearchUserModel[source]

Search for user information based on username and role.

Parameters:
- username (str): The username to search for. Defaults to an empty string, meaning no username filtering is applied.
- page (int): The page number, used to indicate the page of results to return. Defaults to 1, meaning the first page of results is returned.
- page_size (int): The page size, used to specify the number of results per page. Defaults to 20, meaning 20 results per page. - role (str): The user role. Defaults to an empty string, meaning no role filtering is applied.

Available values are 1 = user, 10 = admin, -1 = banned user.

Return:
- SearchUserModel: A model containing the search results. This model includes a list of users that match the search criteria, along with pagination information such as total data count and total pages.

Method to create a new user.

Parameters:
- username (str): The username, of type string.
- password (str): The password, of type string.
- permission (int): The permission level, of type integer, with a default value of 1.

Return:
- str|bool: Returns the user's UUID upon successful creation. If the field is not found, it defaults to returning True.

update(uuid: str, config: dict[str, Any]) → bool[source]

Method to update user information.

It is not recommended to directly use this function. It is advised to call the search method first, then use the update method to update as needed.

Parameters:
- uuid (str): The user's unique identifier (UUID).
- config (dict[str, Any]): The new user information provided as a dictionary. Missing content will be completed by the UserConfig model.

Return:
- bool: Returns True upon successful update.
