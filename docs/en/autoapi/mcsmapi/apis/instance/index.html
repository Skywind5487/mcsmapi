search(daemonId: str, page: int = 1, page_size: int = 20, instance_name: str = '', status: str = '', tag: list[str] | None = None) → mcsmapi.models.instance.InstanceSearchList[source]

Search instance information based on specified parameters.

Parameters:
- daemonId (str): The unique identifier of the daemon process.
- page (int): The page number to indicate which data page to return. Default is 1, representing the first page of data. - page_size (int): The page size to specify how many data items are included per page. Default is 20, representing 20 items per page.
- instance_name (str): The name of the instance. Default is an empty string, meaning no filtering based on the instance name. - status (str): The status of the instance. Default is an empty string, meaning no filtering based on status.
- tag (list[str] | None): The list of tags associated with the instance. Default is None, meaning no filtering based on tags.

Return:
- InstanceSearchList: A model containing the search results. This model includes a list of instance information that matches the search criteria, along with pagination details such as the total number of data items and the total number of pages.
