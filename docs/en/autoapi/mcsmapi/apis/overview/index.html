class mcsmapi.apis.overview.Overview[source]
init()[source]

Initialization method, used to retrieve API overview information and construct an overview model.

This method sends a GET request to fetch the API overview information, ensuring that the returned data type is a dictionary. It then uses this data to construct an OverviewModel instance.

Returns:

Returns an OverviewModel instance, which is initialized using the retrieved API overview information.
